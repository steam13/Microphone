Utilización del
micrófono remoto

Una vez que haya configurado la conexión con Dragon, toque el icono del micrófono para activar el micrófono de Dragon. Ahora ya podrá dictar en Dragon utilizando su dispositivo móvil como micrófono inalámbrico:

Toque de nuevo el micrófono para apagar el micrófono de Dragon:

Para detener temporalmente la escucha del micrófono, diga "A dormir”. Para reactivar el micrófono, diga "A trabajar”. Así es como aparece el micrófono cuando dice "A dormir”:

No puede desactivar el micrófono remoto de Dragon mediante la voz.

Notas: si va a dictar con la aplicación Micrófono remoto de Dragon y va a cambiar a otra aplicación en su teléfono inteligente o dispositivo compatible, Dragon dejará de escucharle. Cuando regrese a la aplicación del micrófono, podrá seguir dictando.
