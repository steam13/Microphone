Configuración del
micrófono remoto

Hay dos formas de configurar la aplicación Micrófono remoto de Dragon para establecer una comunicación con Dragon.

Configuración manual

Al crear o modificar un perfil de Dragon para utilizarlo con la aplicación Micrófono remoto de Dragon, el programa mostrará la información de red necesaria para configurar de forma manual la aplicación Micrófono remoto de Dragon.

Una vez que haya creado o modificado un perfil de usuario para la aplicación Micrófono remoto de Dragon, podrá obtener esta información seleccionando Herramientas > Información micrófono remoto en la barra de Dragon.

1) Pulse el botón Menú y, a continuación, toque el icono de engranaje para acceder a la pantalla Configuración, donde podrá configurar o modificar la conexión de red de Dragon.

2) Toque "Introducción manual” para acceder a la pantalla Introducción manual.

5) Toque el botón del micrófono y, si cambia a color verde, significa que la conexión es correcta.

Utilización de códigos de barras con Dragon

Al crear o modificar un perfil de Dragon para utilizar la aplicación Micrófono remoto de Dragon, Dragon muestra un código de barras en la pantalla del equipo.

Si su dispositivo móvil dispone de cámara, puede utilizar la aplicación Micrófono remoto de Dragon para leer este código de barras y transferir automáticamente los datos de la red a la aplicación Micrófono remoto de Dragon.

Una vez en la pantalla "Introducción manual", toque "Utilizar un código de barras".

El dispositivo móvil le pedirá que coloque el código de barras en la pantalla del equipo, en el visor que se muestra. Una vez que haya colocado el código de barras en el visor, el nombre de perfil de Dragon, la dirección IP y el número de puerto del equipo en el que está instalado Dragon se transferirán automáticamente a la aplicación Micrófono remoto de Dragon.

Una vez capturada toda esta información, toque “Hecho” y la pantalla del micrófono ahora mostrará el nombre del perfil seleccionado.
