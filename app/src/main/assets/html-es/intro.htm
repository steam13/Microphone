Introducción

Requisitos

Para utilizar la aplicación Micrófono remoto de Dragon como micrófono inalámbrico, tanto su dispositivo móvil como el equipo en el que se ejecuta Dragon deben estar en la misma red inalámbrica.

Antes de utilizar la aplicación Micrófono remoto de Dragon, debe crear un perfil de Dragon nuevo o modificar un perfil de Dragon existente para utilizar la aplicación Micrófono remoto de Dragon.

Deberá configurar manualmente la aplicación Micrófono remoto de Dragon.

Consulte la ayuda que se suministra con el software de Dragon para obtener más información.

La primera vez que inicia la aplicación

Toque el icono del micrófono para empezar:

Desde la pantalla Configuración, puede seleccionar “Introducción manual” para introducir los datos sobre el perfil y la red del equipo en el que se ejecuta Dragon.

Mediante la tecla Menú puede realizar lo siguiente:

  • Toque el icono de engranaje para acceder a la pantalla Configuración, donde podrá configurar o modificar la conexión de red de Dragon.
  • Toque el icono de información para acceder a la ayuda.
