Remote-Mikrofon
verwenden

Wenn die Verbindung zu Dragon eingerichtet ist, tippen Sie auf das Mikrofon-Symbol, um das Dragon-Mikrofon einzuschalten. Ihr Mobilgerät ist jetzt ein drahtloses Mikrofon. Sie können jetzt mit dem Diktieren beginnen:

Tippen Sie erneut auf das Mikrofon, um das Dragon-Mikrofon auszuschalten:

Sagen Sie "Geh schlafen" für Dragon NaturallySpeaking oder "Schlaf-Modus" für Dragon Dictate, um die Diktataufnahme vorübergehend auszusetzen. Um das Mikrofon erneut zu aktivieren, sagen Sie "Wach auf" für Dragon NaturallySpeaking oder "Aufwachen" für Dragon Dictate. So wird das Mikrofon nach dem Sprachbefehl "Geh schlafen" (Dragon NaturallySpeaking) oder "Schlaf-Modus" (Dragon Dictate) dargestellt:

Sie können das Dragon Remote-Mikrofon nicht mit Ihrer Stimme ausschalten.

Hinweis: Wenn Sie mit der Dragon-Anwendung für Remote-Mikrofon diktieren und auf Ihrem unterstützten Gerät zu einer anderen Anwendung wechseln, unterbricht Dragon die Mikrofonaufnahme. Wenn Sie zur Mikrofon-Anwendung zurückkehren, können Sie mit dem Diktieren fortfahren.
