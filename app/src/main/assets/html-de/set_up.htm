Remote-Mikrofon
einrichten

Es gibt zwei Möglichkeiten, die Dragon-Anwendung für Remote-Mikrofon für die Kommunikation mit Dragon zu konfigurieren.

Manuell einrichten

Wenn Sie ein Benutzerprofil für die Arbeit mit der Dragon-Anwendung für Remote-Mikrofon erstellen oder ändern möchten, zeigt Dragon die Netzwerkinformationen an, die Sie benötigen, um die Dragon-Anwendung für Remote-Mikrofon manuell zu konfigurieren.

Nachdem Sie auf Ihrem Computer ein Benutzerprofil für die Nutzung der Dragon-Anwendung für Remote-Mikrofon eingerichtet oder angepasst haben, können Sie diese Angaben auf Ihrem Computer auf der Dragon-Leiste im Menü Extras > Informationen zum Remote-Mikrofon abrufen.

1) Drücken Sie auf die Menü-Schaltfläche und tippen Sie auf das Zahnrad-Symbol , um die “Einstellungen” anzuzeigen und die Netzwerkverbindung mit Dragon einzurichten oder zu ändern.

2) Tippen Sie auf "Manuell Eingeben", um den Bildschirm für die manuelle Eingabe zu öffnen.

3) Geben Sie unter Verwendung der von Dragon gelieferten Informationen den Dragon-Profilnamen, die IP-Adresse sowie die Portnummer des Computers ein. Beispiel:

Tippen Sie dann auf “Fertig”.

5) Tippen Sie auf die Schaltfläche “Mikrofon”. Wenn diese grün angezeigt wird, haben Sie erfolgreich eine Verbindung hergestellt.

Barcodes mit Dragon benutzen

Wenn Sie ein Dragon-Profil erstellen oder ändern, um die Dragon-Anwendung für Remote-Mikrofon zu verwenden, zeigt Dragon einen Barcode auf dem PC-Bildschirm an.

Wenn Ihr Mobilgerät über eine Kamera verfügt, können Sie diesen Barcode mithilfe der Dragon-Anwendung für Remote-Mikrofon ablesen und die Netzwerkdaten automatisch an die Anwendung übertragen lassen.

Wenn Sie den Bildschirm "Manuell" geöffnet haben, tippen Sie auf "Barcode verwenden".

Sie werden von Ihrem Mobilgerät aufgefordert, den am Computerbildschirm angezeigten Barcode mit dem Suchfenster aufzunehmen. Wenn sich der Barcode im Suchfenster befindet, werden der Dragon-Profilname, die IP-Adresse und die Portnummer des Computers, auf dem Dragon installiert ist, automatisch an die Dragon-Anwendung für Remote-Mikrofon übertragen.

Nachdem die Informationen erfasst wurden, tippen Sie auf "Fertig". Im Bildschirm "Mikrofon" wird nun der Name des ausgewählten Profils angezeigt.
