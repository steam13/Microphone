Erste Schritte

Anforderungen

Damit Sie die Dragon-Anwendung für Remote-Mikrofon als drahtloses Mikrofon verwenden können, müssen Ihr Mobilgerät und der Computer, auf dem Ihre Dragon-Software installiert ist, auf demselben drahtlosen Netzwerk eingerichtet sein.

Bevor Sie die Dragon-Anwendung für Remote-Mikrofon verwenden können, müssen Sie auf Ihrem Computer ein entsprechendes Dragon-Profil erstellen oder ändern.

Sie müssen die Dragon-Anwendung für Remote-Mikrofon manuell konfigurieren.

Weitere Informationen finden Sie in der Dragon-Hilfe.

Beim ersten Start der Anwendung

Tippen Sie auf das Mikrofon-Symbol, um zu beginnen:

Unter "Einstellungen" können Sie die Option "Manuell eingeben" auswählen, um die Netzwerk- und Profilinformationen für den Computer einzugeben, auf dem Dragon ausgeführt wird.

Über die "Menü"-Taste können Sie:

  • Auf das Zahnradsymbol tippen, um “Einstellungen” aufzurufen. Hier richten Sie die Netzwerkverbindung zu Dragon ein oder passen sie an.
  • Auf das Informationsysmbol tippen, um die Hilfe aufzurufen.
