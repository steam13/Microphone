Getting Started

Requirements

To use the Dragon Remote Microphone Application as a wireless microphone, both your mobile device and the computer where Dragon is running must be on the same wireless network.

Before you can use the Dragon Remote Microphone Application, you must create a new Dragon profile or modify an existing Dragon profile to use the Dragon Remote Microphone Application.

You will need to manually configure the Dragon Remote Microphone Application.

Refer to the Help that came with your Dragon software to get more information.

The first time you start the app

Tap the microphone icon to get started:

From the Settings screen, you can select "Manual Entry" to enter the network and profile information for the computer where Dragon is running.

The Menu Key brings up a menu where you can:

  • Tap the gears icon to get to the Settings screen to setup or modify the network connection to Dragon.
  • Tap the information icon to get the help.
