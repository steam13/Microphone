Set Up the
Remote Microphone

There are two ways to configure the Dragon Remote Microphone Application to communicate with Dragon.

Manual setup

When you create or modify a Dragon profile for use with the Dragon Remote Microphone Application, Dragon will display the network information you need to manually configure the Dragon Remote Microphone Application.

Once you have created or modified a user profile for the Dragon Remote Microphone Application, you can get this information by selecting Tools > Remote Microphone Info from the DragonBar.

1) Press the Menu Button and then Tap the gears icon to get to the Settings screen to setup or modify the network connection to Dragon.

2) Tap "Manual Entry" to display the Manual Entry screen.

3) Using the information supplied by Dragon, enter the Dragon Profile Name, the IP Address, and the computer port number. For example:

When you are done entering the information, tap "Done."

4) The microphone screen will now display the name of the Profile you selected. For example:

5) Tap the microphone button and if it turns green, you have a successful connection.

Using barcodes with Dragon

When you create or modify a Dragon profile to use the Dragon Remote Microphone Application, Dragon displays a barcode on your computer screen.

If your mobile device has a camera, you can use the Dragon Remote Microphone Application to read this barcode and automatically transfer the network data to the Dragon Remote Microphone Application for you.

When you are on the "Manual Entry" screen, tap "Use a barcode."

Your mobile device will ask you to place the barcode on your computer screen inside the displayed viewfinder. When the barcode is in the viewfinder, the Dragon Profile Name, the IP Address, and the computer port number of the computer where Dragon is installed will be automatically transferred to the Dragon Remote Microphone Application.

Once that information is captured, tap "Done" and the microphone screen will now display the name of the Profile you selected.
