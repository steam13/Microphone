Remote Microphone の
使用

Dragon との接続を設定したら、マイクロフォン アイコンをタップして Dragon Remote Microphone を起動します。 これで、モバイル機器をワイヤレス マイクとして使用し Dragon で音声入力を行う準備が整いました。

マイクロフォンを再度タップして Dragon Remote Microphone をオフにします。

マイクロフォンを一時停止するには、「スリープ」などと言い、 「スリープ解除」などと言うと再起動します。 マイクロフォンが「スリープ」を認識すると、以下のように表示されます。

音声で Dragon Remote Microphone をオフにすることはできません。

メモ: スマートフォンやその他の対応デバイスで、Dragon Remote Microphone アプリケーションの使用中にその他のアプリケーションに切り替えると、Dragon は音声入力を停止します。 その後マイクロフォン アプリに戻ると、音声入力が再開されます。
