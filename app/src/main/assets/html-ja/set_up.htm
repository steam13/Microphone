Dragon Remote Microphone アプリケーションと Dragon の通信を設定する方法は2通りあります。

手動で設定する

まず準備として、Dragon が起動しているコンピュータのネットワーク情報を取得する必要があります。

Dragon Remote Microphone アプリケーションで使用するために Dragon プロファイルを作成、あるいは音声機器の追加によってプロファイルを変更すると、その過程で Dragon は Dragon Remote Microphone アプリケーションを手動で設定するために必要なネットワーク情報を表示します。

また、Dragon Remote Microphone アプリケーション用のユーザー プロファイルを作成あるいは変更した後は、Dragon バーの [ツール] から [Remote Microphone 情報] を選択して、情報を取得することもできます。

1) メニュー キーを押し、ギア アイコン をタップして [設定] 画面を開き、Dragon へのネットワーク接続の設定や変更を行います。

2) [手動入力] をタップすると [手動入力] 画面が表示されます。

Dragon の QR コードを使用する

Dragon Remote Microphone アプリケーションで使用するために Dragon プロファイルを作成、あるいは音声機器の追加によってプロファイルを変更すると、その過程で Dragon は画面にQR コードを表示します。

お使いのモバイル機器にカメラが搭載されている場合は、Dragon Remote Microphone アプリケーションを使ってこのQR コードを読み取ると、Dragon Remote Microphone アプリケーションにネットワーク データが自動転送されます。

読み取りを行うには[手動入力] 画面で [QR コードの使用] をタップします。

モバイル機器に、コンピュータ画面の QR コードを表示されたファインダー内におさめスキャンするよう求めるメッセージが表示されます。QR コードがファインダーに入ると、Dragon プロファイル名および IP アドレス、Dragon をインストールしたコンピュータのポート番号が、Dragon Remote Microphone アプリケーションに自動転送されます。

情報取得の完了後 [完了] をクリックすると、マイク画面に現在選択しているプロファイルの名前が表示されます。
