Configurer le
Microphone à distance

Deux méthodes permettent de configurer l'application Microphone à distance pour communiquer avec Dragon.

Configuration manuelle

Lorsque vous créez ou modifiez un profil utilisateur pour l'utiliser avec l'application Microphone à distance, Dragon affiche les paramètres réseau nécessaires à la configuration manuelle de Microphone à distance.

Une fois que vous avez créé ou modifié un profil utilisateur dans l'application Microphone à distance, vous pouvez obtenir ces informations en sélectionnant Outils > Informations pour le micro à distance dans la barre Dragon.

1) Sélectionnez le bouton Menu, puis l'icône engrenages pour accéder à l'écran Paramètres pour configurer ou modifier la connexion réseau avec Dragon.

2) Sélectionnez Saisie manuelle pour afficher l'écran Saisie manuelle.

3) À l'aide des informations fournies par Dragon, entrez le nom du profil de Dragon, l'adresse IP et le numéro du port de l'ordinateur. Par exemple :

Lorsque vous avez terminé la saisie des informations, appuyez sur Terminé.

Utiliser un code-barres avec Dragon

Lorsque vous créez ou modifiez un profil Dragon pour utiliser l'Aplication Microphone à distance de Dragon, Dragon affiche un code-barres sur l'écran de l'ordinateur.

Si votre appareil mobile dispose d'un appareil photo, vous pouvez utiliser l'Aplication Microphone à distance de Dragon pour lire ce code-barres et transférer automatiquement les données du réseau à l'Aplication Microphone à distance de Dragon.

Dans l'écran Saisie manuelle, appuyez sur la touche Utiliser un code-barres.

Votre appareil mobile vous demande alors de placer le code-barres sur l'écran de l'ordinateur à l'intérieur du viseur qui s'affiche. Une fois le code-barres dans le viseur, le nom de profil de Dragon, l'adresse IP et le numéro de port de l'ordinateur sur lequel Dragon est installé seront automatiquement transférés à l'Aplication Microphone à distance de Dragon.

Une fois ces informations capturées, appuyez sur la touche Terminé. L'écran du microphone affiche à présent le nom du profil sélectionné.
