Mise en route

Configuration requise

Pour utiliser l'application Microphone à distance comme micro sans fil, votre appareil portatif et l'ordinateur où est exécuté Dragon doivent figurer sur le même réseau sans fil.

Avant de pouvoir utiliser l'application Microphone à distance, vous devez créer ou modifier un profil Dragon.

Vous devrez configurer manuellement l'application Microphone à distance.

Pour plus d'informations, consultez l'Aide fournie avec votre logiciel Dragon.

Première utilisation de l'application

Sélectionnez l'icône du microphone pour démarrer :

Dans l'écran Paramètres, vous pouvez sélectionner Saisie manuelle pour renseigner les informations de réseau et de profil de l'ordinateur sur lequel est exécuté Dragon.

La touche Menu vous propose les options suivantes :

  • Sélectionner l'icône des engrenages pour accéder à l'écran Paramètres afin de configurer ou de modifier la connexion réseau à Dragon.
  • Sélectionner l'icône d'information pour obtenir de l'aide.
