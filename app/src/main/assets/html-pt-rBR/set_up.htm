Configurar o
Remote Microphone

Existem duas formas de configurar o Dragon Remote Microphone Application para se comunicar com o Dragon.

Configuração manual

Quando você criar ou modificar um perfil do Dragon para usá-lo com o Dragon Remote Microphone Application, o Dragon exibirá as informações de rede necessárias para a configuração manual do Dragon Remote Microphone Application.

Depois que você tiver criado ou modificado um perfil de usuário do Dragon Remote Microphone Application, poderá obter essas informações selecionando Ferramentas > Informações sobre o Remote Microphone na barra Dragon.

1) Pressione o botão Menu e toque no ícone de engrenagem para abrir a tela Configurações para configurar ou modificar a conexão de rede com o Dragon.

2) Toque em "Entrada manual" para exibir a tela Entrada manual.

5) Toque no botão do microfone se ele ficar verde, pois a conexão foi estabelecida.

Usando os códigos de barras do Dragon

Quando você cria ou modifica um perfil para usar o Dragon Remote Microphone Application, o Dragon exibe um código de barras na tela do computador.

Se o seu dispositivo móvel possui uma câmera, você poderá usar o Dragon Remote Microphone Application para ler o código e transferir os dados da rede automaticamente.

Quando você estiver na tela "Entrada manual", toque em "Usar um código de barras".

O seu dispositivo móvel solicitará que você posicione o código na tela do computador dentro do visor exibido. Quando o código de barras estiver no visor, o Dragon Profile Name, o Endereço IP e o número de porta do computador onde o Dragon está instalado serão transferidos automaticamente para o Dragon Remote Microphone Application.

Quando essa informação for obtida, toque em “Concluir”, e a tela do microfone exibirá o nome do perfil que você selecionou.
