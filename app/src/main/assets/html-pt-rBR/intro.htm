Noções básicas

O Dragon Remote Microphone Application permite que você use seu dispositivo móvel como um microfone sem fio com:

  • Dragon NaturallySpeaking V11.5 ou superior
  • Dragon Dictate V2.5 ou superior

Requisitos

Para usar o Dragon Remote Microphone Application como um microfone sem fio, seu dispositivo móvel e o computador no qual o Dragon está sendo executado devem estar na mesma rede sem fio.

Antes de usar o Dragon Remote Microphone Application, você precisará criar um novo perfil no Dragon ou modificar o perfil existente para usar o Dragon Remote Microphone Application.

Você precisará configurar manualmente o Dragon Remote Microphone Application.

Consulte a Ajuda fornecida com o software Dragon para obter mais informações.

Na primeira vez que iniciar o aplicativo

Toque no ícone do microfone para começar:

Na tela Configurações, selecione Entrada manual para inserir manualmente as informações da rede e do perfil do computador no qual o Dragon está sendo executado.

A tecla Menu abre um menu no qual é possível:

  • Tocar no ícone de engrenagem para abrir a tela Configurações para configurar ou modificar a conexão de rede com o Dragon.
  • Tocar no ícone de informações para acessar a Ajuda.
