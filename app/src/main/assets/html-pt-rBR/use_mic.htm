Usando o
Remote Microphone

Depois de configurar uma conexão com o Dragon, toque no ícone do microfone para ativar o microfone do Dragon. Você poderá ditar para o Dragon usando seu dispositivo móvel como um microfone sem fio:

Toque no microfone novamente para desativar o microfone do Dragon:

Para silenciar o microfone temporariamente, diga "Entrar em espera". Para ativar novamente o microfone, diga "Ativar". Esta é a aparência do microfone quando você diz "Entrar em espera":

Não é possível desligar o Dragon Remote Microphone com a voz.

Notas: Se você estiver ditando com o Dragon Remote Microphone Application e alternar para outro aplicativo no seu smartphone ou em um dispositivo compatível, o Dragon não ouvirá mais o que você está dizendo. Quando você retornar para o aplicativo do microfone, poderá continuar ditando.
